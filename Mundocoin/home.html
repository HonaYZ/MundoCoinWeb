using blockchain technology to enable a fast,
secure and scalable trading experience.

Faq

MundoCoin/MNO is a new L1 Coin, Is an open source digital currency project developed based on the PolyKerma kernel that uses blockchain technology to enable a fast, secure and scalable transaction experience. Aims to create a secure, fast, community-engaged cryptocurrency ecosystem.
MundoCoinNode/MCN is a credential, and only with MCN can you run the master node,It is also a security measure to prevent malicious nodes from attacking the network.
Total supply is 21,000,000. Current supply can be found on MNO blockchain explorer.
MNO is not yet listed on exchanges, but it will be soon.
Can use PC wallet, mobile wallet to store and send MNO.
You can choose to run the master node of the MNO through the server, or run the mini node through the mobile phone to obtain the MNO.
